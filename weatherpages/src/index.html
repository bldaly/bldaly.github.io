    • The purpose of this assignment was to analayze how weather changes the closer that you get to the equator. To accomplish this analysis, we used we pulled data from OpenWeatherMap API on a dataset of over 500 cities.

    After the dataset was assembled, we used Matplotlib to graph the various findings. Factors we looked at included the following: temperature, cloudiness, wind speed, and humidity. This site includes the data as well as the visualizations created as part of the analysis.
